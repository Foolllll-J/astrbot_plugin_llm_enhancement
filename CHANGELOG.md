# Changelog

## 1.1.0

### Added

- **提及规则作用域限定**：规则后支持 `@g:群号` 与 `@u:用户ID` 限定生效范围，加 `!` 前缀表示排除
- **唤醒判定模板占位符**：新增 `{rN}` 随机数与 `{schedule}` 日程占位符
- **群聊按用户并发（`group_sender_concurrency`）**：同群不同用户的 LLM 请求各自独立排队，不再全局串行等待
- **预合并窗口配置（`merge_premerge_window`）**：合并发起前预留缓冲窗口以进一步聚合消息
- **B站 API 元数据解析**：自动提取视频标题、UP 主、时长、播放/弹幕/点赞/评论数，注入到上下文
- **LLM 语音转写 fallback**：STT 引擎不可用时自动回退到 LLM ASR 进行语音识别（`asr_llm_provider_id` 配置项）
- **Tavily extract URL 解析**：URL 解析新增 Tavily extract 作为网页内容提取引擎
- **引用文本段 URL 解析**：引用消息正文中的链接自动走 B站元数据解析或网页解析流程
- **QQ 表情 ID 映射**：补充更多官方表情 ID → 含义映射
- **Bot 身份注入**：`is_robot` 字段标记消息来源是否为 Bot 自身
- **非 LLM 回复补录**：框架直接回复的消息自动补录到群聊上下文，保持对话连贯性
- **send_qq_message 补录**：通过工具跨群发送的消息写入目标群的上下文记录
- **自我操作限制配置（`self_only_tools`）**：非管理员调用所选工具时只能作用于自己
- **黑名单表格图片输出**：`list_blacklist` 支持以图片表格形式展示结果，支持批量拉黑

### Changed

- **合并与并发配置重构**：合并策略、窗口控制、并发上限统一归入 `request_orchestration` 配置段
- **上下文注入改为后台任务**：不阻塞主消息处理流程
- **上下文注入行宽上限提升**：`_render_context_line` limit 从 110 提高到 130
- **图片转述提示词升级为详细版**：120 字约束，覆盖 7 个分析维度
- **send_message 重命名为 send_qq_message**：使工具名更准确反映功能
- **SubAgent 建议文案更新**：黑名单工具描述、拉黑规则、输出规则调整
- **统一中文数字表达**：如"一条"→"1条"、"一个"→"1个"等
- **统一平台检测方法**：封装为 `_get_aiocqhttp_client`，消除重复的平台判断逻辑

### Removed

- **blocked_commands 屏蔽指令功能**：不再通过插件维护指令屏蔽，由框架权限系统接管
- **security_blacklist Agent**：合并到 QQ Expert Agent，统一管理工具调用策略
- **tool_admin_required_tools 配置**：由框架内置的运行时工具权限系统替代
- **Pillow 依赖**：框架已内置该依赖，插件无需重复要求

### Fixed

- **合并上限误拦截**：达到合并数量上限时错误地阻止了后续正常请求
- **动态合并失效**：预合并快照未正确附着导致合并流程中断
- **intercept_level 配置顺序错乱**：黑名单拦截等级选项顺序与预期不一致
- **url_parsing 配置未生效**：URL 解析开关配置不生效的问题

### Performance

- **URL 解析增强**：改进正文提取、Cloudflare 绕过、meta description 兜底、错误过滤
- **视频 ASR + 逐帧识图并发**：语音转录与关键帧分析同时进行，减少等待时间
- **动态 followup 清理**：及时清理残留的 followup 对话历史
- **丢弃响应不记录**：`should_drop` 标记的响应不再写入上下文
- **合并上限降级**：达到合并上限时降级为新消息而非丢弃
- **合并缓冲区去重**：相同内容的消息自动压缩，避免重复处理
- **图片转述超时放宽**：超时从 20s 增加到 60s，`timeout_sec=0` 时无等待
- **图片压缩**：超过 2048px 的图片自动 LANCZOS 缩小并 JPEG 95 压缩
- **预合并流程优化**：减少不必要的上下文快照与重算
- **唤醒屏蔽流程优化**：减少早期判断阶段的非必要计算
- **context_messages 按时间戳排序**：确保上下文顺序稳定可预测
- **成员信息过滤**：过滤群成员信息中的无意义空值字段
- **汇总摘要超时缩短**：从默认降至 15s，减少长时间无响应的情况

### Docs

- README 更新：提及规则作用域、工具重命名、Agent 合并、自我操作限制、请求编排与并发控制

## 1.0.0

### Added

- **智能唤醒系统**：概率唤醒（根据活跃度动态调整触发门槛）、空 @ 唤醒（自动注入最近群聊上文）、提及唤醒（支持 regex 与文本匹配）、相关性唤醒（自动判断话题延续性）、专业答疑（两段式判定）、无聊唤醒、唤醒节奏优化（降权+沉默补偿）
- **唤醒判定链路**：前置唤醒判定（独立模型/提示词）、后置唤醒兜底（按唤醒类型过滤）、活跃在线期、同用户唤醒延长、唤醒延长 T/F 判定
- **并发与约束**：用户级与群聊级并发上限、唤醒屏蔽（文本包含 / 正则 / 指令）、唤醒必须 @Bot、机器人消息跳过唤醒
- **消息合并**：动态软重算与硬等待合并两种模式、多人消息聚合、合并数量上限、撤回实时剔除、请求前/回复前最终校验
- **媒体解析**：私聊媒体唤醒（表情/视频/文件/语音/JSON 组件）、QQ 黄脸表情含义解析、视频抽帧 + ASR、语音 ASR 转写、文本/PDF 文件片段注入、JSON 卡片提取、链接解析（标题/正文/超时/域名黑名单）、合并转发完整解析（含视频/图片/卡片）、发送者身份注入、引用 Bot 自身消息智能屏蔽、视频/转发总结缓存
- **上下文注入**：独立群聊上下文记录链路（主动唤醒前/全唤醒前/关闭）、通知/申请事件记录、精简/详细两种注入格式、URL/JSON/语音/文件/聊天记录抽取能力
- **环境感知注入**：平台、会话类型、群名、群号、时间、星期、节假日
- **群成员信息注入**：用户 ID、昵称、群名片、备注、性别、年龄、群等级、群身份、群头衔、入群时间、生日、地区、签名、标签、Q龄、QQ 等级、会员信息
- **LLM 工具**：8 个查询类工具（头像/通讯录/群成员/群信息/公告/精华/历史消息）、13 个群管理写操作（禁言/踢人/改名/公告/精华/撤回/解散等）、黑名单管理（拉黑/解除/分页查询/状态查询）
- **工具权限**：管理员校验、危险工具开关、二次确认
- **SubAgent**：QQ 平台专家（qq_expert）与黑名单安全策略助手（security_blacklist）两套推荐配置
